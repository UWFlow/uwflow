# API server

This is the Flow API server for handling non-CRUD tasks.

## Features

- Authentication:
  - [x] email
  - [x] openID
- Parsing:
  - [x] schedule
  - [x] transcript
- [x] Webcal generation

## How to run this

As per `Dockerfile`, this is containerized and managed by `docker-compose`.
If you wish to run this on the host, do so at your peril.
